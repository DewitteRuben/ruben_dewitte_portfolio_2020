* Strongly typed JavaScript with <MdLink href="https://www.typescriptlang.org/">Typescript</MdLink> 
* UI and styling with <MdLink href="https://styled-components.com/">Styled Components</MdLink>
* <MdLink href="https://formspree.io/">Formspree</MdLink> for the contact form
